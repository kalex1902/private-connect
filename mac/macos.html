Mac — подключение

Установка займёт 2–3 минуты

Шаг 1

Скачать приложение

Откройте страницу загрузки
Нажмите «Download Tailscale for macOS»
Файл появится в загрузках
Шаг 2

Установка

Откройте файл
Нажмите «Продолжить»
Нажмите «Установить»
Введите пароль
Дождитесь завершения
Нажмите «Закрыть»
Шаг 3

Разрешения

Нажмите OK
Нажмите Install Now
Откройте настройки
Включите Tailscale
Шаг 4

Подключение

Разрешите подключение
Подтвердите действие
Шаг 5

Аккаунт

Откройте настройки
Добавьте аккаунт
Введите адрес сервера
Подтвердите вход
Шаг 6

Готово

Пропустите обучение
Убедитесь, что статус — Connected
🎉 Всё готово — можно пользоваться
