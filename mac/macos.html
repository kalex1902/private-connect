💻 Mac — установка Tailscale

Следуйте инструкции — это займет 2–3 минуты

📥 Шаг 1. Скачать приложение

Откройте сайт и скачайте Tailscale
Скачать Tailscale
Файл появится в загрузках

⚙️ Шаг 2. Установка

Откройте скачанный файл
Нажмите «Продолжить»
Нажмите «Установить»
Введите пароль от Mac
Дождитесь завершения установки
Нажмите «Закрыть»
Переместите установщик в корзину

🔐 Шаг 3. Разрешения

Обязательный шаг — без этого ничего работать не будет
Нажмите OK
Нажмите Install Now
Нажмите Open Settings
Включите Tailscale
Введите пароль
Нажмите «Готово»

🌐 Шаг 4. Разрешить подключение

Нажмите Allow Configuration
Подтвердите действие

👤 Шаг 5. Добавить аккаунт

Откройте настройки аккаунта
Нажмите Add Account
Введите адрес сервера и нажмите Add Account
Пройдите авторизацию в браузере

🔌 Шаг 6. Подключение

Пропустите обучение (Skip)
Убедитесь, что статус — Connected

⚙️ Шаг 7. Настройки (рекомендуется)

Включите необходимые параметры
🎉 Готово! Всё работает
Теперь устройство подключено и готово к работе
Если что-то не работает — напишите тому, кто вас подключал
