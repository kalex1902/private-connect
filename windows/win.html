💻 Windows — подключение

Mesh-сеть T-Guard · установка займёт 2–3 минуты

Шаг 1. Скачать приложение

⬇ Скачать Tailscale Открыть сайт (если не скачалось)
После скачивания:
1. Откройте файл
2. Нажмите Next → Next → Install
Если файл не скачался — попробуйте Chrome или Edge

Шаг 2. Подключение

⚡ Скачать и подключиться
1. Скачайте архив
2. Откройте его
3. Перетащите файл на рабочий стол
4. Запустите файл "Подключиться к T-Guard.bat" (двойным кликом)
Если Windows ругается → «Подробнее» → «Выполнить в любом случае»
💡 Это нормально — файл безопасен

Шаг 3. Авторизация

Войдите через свой Google-аккаунт (он будет использоваться на всех устройствах)
Дождитесь "Sign in successfully"

📡 Подключение

1. Откройте Tailscale
2. Включите переключатель
3. Выберите любую страну

💡 Интернет начнёт работать через выбранную страну
